## Quick Installation

```bash
pip install agenticaiframework
```

## Minimal Example

```python
from agenticaiframework import Agent, AgentManager

# Create an agent
agent = Agent(
    name="MyFirstAgent",
    role="Assistant",
    capabilities=["text_generation"],
    config={}
)

# Manage agents
manager = AgentManager()
manager.register_agent(agent)

# Start the agent
agent.start()
print(f"Agent {agent.name} is {agent.status}")
```
